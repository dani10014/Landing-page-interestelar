🌌 Landing Page - Interestelar

"O amor é a única coisa que somos capazes de perceber que transcende as dimensões do tempo e do espaço."

Este projeto é uma landing page temática dedicada ao filme Interestelar de Christopher Nolan. O objetivo foi criar uma interface imersiva que apresenta os personagens principais, o enredo e as plataformas onde a obra pode ser assistida, utilizando tecnologias modernas de front-end.
🚀 Tecnologias e Conceitos Aplicados

    HTML5 Semântico: Estruturação focada em acessibilidade e SEO.

    CSS3 Avançado & Flexbox: Utilização de layouts flexíveis para garantir que a página seja responsiva.

    Efeitos de Transição: Implementação de efeitos de hover e interatividade nos cards dos personagens.

    Bootstrap 5: Uso de componentes como o Carrossel para navegação de imagens e classes utilitárias para agilizar o desenvolvimento.

    Design Imersivo: Escolha de paleta de cores e tipografia que remetem à estética espacial e tecnológica do filme.

🛠️ Funcionalidades

    Seção de Personagens: Cards detalhados com informações sobre Cooper, Murph, Professor Brand, TARS, Amelia e Tom.

    Guia de Streaming: Lista atualizada de plataformas (Prime Video, HBO Max, Claro tv+) com informações de planos e preços.

    Formulário de Contacto: Área dedicada para os utilizadores sugerirem novos filmes, utilizando classes de formulário do Bootstrap.

    Interface Responsiva: Design adaptado para diferentes tamanhos de ecrã, garantindo uma boa experiência tanto no desktop quanto no mobile.
